Welcome to my portfolio. My name is Eirik Kirkeli and on this page i will display my journy from beginner to proffesional web developer. I am a student at Noroff Vocational College in Norway. I started my jurney in 2022 as a beginner, and invite you to see my journy into the fantastic world of web development.

Trollmagi

Trollmagi was my year one exam project. The task was to create a blog by fetching from a WP API. I created a blog about the norwegian folklore creature "Troll". I used HTML, CSS, JS and PHP to create this blog. I also used the WP REST API to fetch the data from the WP site. The theme was inspired by the stories i tel my kids befor bed.

Rainy Days

Rainy days was our cross course project. the first project we were to use JS. The brief was to create a webshop for a fictional company. We were to use the WooCommerce API to fetch the data. I used HTML, CSS, JS and PHP to create this webshop.

Community Science Museum

Our first real project in year one. The site is built with HTML and CSS only. The task was to create a website for a fictional science museum. Images and text was provided in the brief. I also contacted a real science museum to get some inspiration and got permision to use some of their content.
