Welcome to my portfolio. My name is Eirik Kirkeli and on this page, I will display my journey from beginner to professional web developer. I am a student at Noroff Vocational College in Norway. I started my journey in 2022 as a beginner and invite you to see my journey into the fantastic world of web development.

Trollmagi

Trollmagi was my year one exam project. The task was to create a blog by fetching from a WP API. I created a blog about the Norwegian folklore creature “Troll”. I used HTML, CSS, JS, and PHP to create this blog. I also used the WP REST API to fetch the data from the WP site. The theme was inspired by the stories I tell my kids before bed.

Rainy Days

Rainy Days was our cross-course project, the first project where we were to use JS. The brief was to create a webshop for a fictional company. We were to use the WooCommerce API to fetch the data. I used HTML, CSS, JS, and PHP to create this webshop.

Community Science Museum

The cross course project exam in year one was a significant milestone. The site was built using only HTML and CSS. The task was to create a website for a fictional science museum. Images and text were provided in the brief. I also contacted a real science museum for inspiration and received permission to use some of their content.
