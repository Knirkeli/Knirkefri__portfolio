Welcome to my portfolio. My name is Eirik Kirkeli and on this page I will display my journey from beginner to professional web developer. I am a student at Noroff Vocational College in Norway. SalmonBeds

My final exam at Noroff!
SalmonBeds is an accommodation site built using Next, Tailwind and Typescript. It uses the Noroff Holidaze api. The user can rent and/or rent out venues. Login, is handled using Cookies.

Bubble Commerce

Buuble Commerce is a e-commerce site built with next, tailwind and typescript using the Noroff online shop API. Users can browse products, add them to their cart and checkout. This was my first project using Next.js and Tailwind CSS.

Check it out

TT-Auctions

TT-Auctions is an auction site built with HTMl, Bootstrap SASS and javascript. Using the Noroff Auction API we were to let users Create auctions and bid on other users auctions. This Site is currently being Rebuilt with Next and Tailwind.

Trollmagi

Trollmagi was my year one exam project. The task was to create a blog by fetching from a WP API. I created a blog about the Norwegian folklore creature “Troll”. I used HTML, CSS, JS, and PHP to create this blog. I also used the WP REST API to fetch the data from the WP site. The theme was inspired by the stories I tell my kids before bed.

Check it out

Rainy Days

Rainy Days was our cross-course project, the first project where we were to use JS. The brief was to create a webshop for a fictional company. We were to use the WooCommerce API to fetch the data. I used HTML, CSS, JS, and PHP to create this webshop.
